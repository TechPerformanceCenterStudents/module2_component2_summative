# Programming Fundamentals Assessment  
**Module 2 – Component 2 (Units 26–30)**  
**Formative & Summative Assessment**

## Overview
Assesses your knowledge of:
- Numeric types  
- Arithmetic  
- Strings & Unicode  
- Type conversion  
- Boolean logic  
- Truthy/falsy values  
- Comparison operators  
- JS conditionals  

## Assessment Structure

### Section A — Multiple Choice
Covers:
- Numeric bases  
- Strings  
- Truthiness  
- Equality pitfalls  

### Section B — Short Answer
Explain:
- Number vs BigInt  
- Why decimals round incorrectly  
- Template literals  
- Three falsy values  
- == vs ===  

### Section C — Coding Tasks

#### 1. Arithmetic
Variables a = 12, b = 5  
Print: sum, remainder, exponent result.

#### 2. Strings & Type Conversion
- String with decimal  
- Number  
- Convert both  
- Round to 2 decimals using `.toFixed(2)`  
- Output with template literals  

#### 3. Boolean Logic & Conditions
Using:
```
let age = 17;
let hasId = false;
let isMember = true;
```
Print one:
- Full access  
- Adult access  
- Member access  
- Access denied  

## Tools

node filename.js
